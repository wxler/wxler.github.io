Anaconda与jupyter安装、操作及插件安装

发表于2020-11-24更新于2020-11-24字数统计695阅读时长5分

一、anaconda简介

  1. 进入官网https://www.anaconda.com
  2. 点解download进入下载页面
  3. 按步骤安装(默认，下一步)
  4. 启动jupyterlab应用

二、jupyter lab 基本操作

1. jupyter lab 简介

  • JupyterLab包含了Jupyter Notebook所有功能。
  • JupyterLab作为一种基于web的集成开发环境，你可以使用它编写notebook、操作终端、编辑markdown文本、打开交互模式、查看csv文件及图片等功能

2. jupyter lab 基本操作

2.1 操作模式

两种操作模式 command mode 和 edit mode

在一个cell中按下enter就进入edit mode，按下Esc进入command mode

2.2 cell转换类型

  • code:代码环境
  • markdown: 带有latex公式输入的增强markdown
  • raw：纯文本

cell类型转换方式

  • 鼠标操作
  • 快捷键

command模式下：m:markdown r:raw text y:code

2.3 运行cell

  • shift+enter:运行并跳转到下一个cell
  • control+enter：运行并停留在当前cell

2.4 增加/删除cell

  • 鼠标操作：两种模式均可
  • 快捷键：a/d command模式
  1. A 在当前cell的上面添加cell
  2. B 在当前cell的下面添加cell
  3. 双击D 快速删除当前cell

2.5 代码提示功能

  • tab 代码补全或缩进
  • Shift-Tab 提示
  • 更多快捷键参照 Jupyter Notebook骚操作

2.6 注释多行代码

  • ctrl / 注释多行代码

2.7 Markdown标题级别

  • 命令行模式下，1 2 3 4 5 6分别是1-6级标题

2.8 cell加行号

  • 命令行模式下，shilt+L为所有cell加上行号，L不持支给单个cell加行号
  • 鼠标点击view显示行号

2.9 cell合并分割

  • 编辑模式，crtl shift - 在光标处分割cell
  • 命令行模式 shift 鼠标 选中多个cell
    shift m 合并选中给的cell

三、插件安装

  • 更新插件：jupyter labextensio update 插件名
  • 更新所有插件：jupyter labextension update --all
  • 卸载插件：jupyter labextensio uninstall 插件名
  • 安装插件：jupyter labextensio install 插件名
  • 远程仓库安装插件：jupyter labextension install 参考地址
  • 安装制定版本插件：jupyter labextensio install 插件名=版本号
  • 查看已安装插件：jupyter labextension list
